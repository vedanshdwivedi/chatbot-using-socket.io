- Install "npm i express ejs body-parser --save"

Dependable: 
- It is a minimalist injection framework for javascript
- This allows for the creation of modules that can be passed as dependencies instead of requiring them [frequently used modules]
- The container.js file will have a dependable container with frequently used modules
